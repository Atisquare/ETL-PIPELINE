# ETL-PIPELINE
ETL Pipeline Demo – OpenWeather API  This video demonstrates my Week 7 Data Pipelines &amp; Automation project completed during the AnalystLab Africa Data Analytics Internship.  In this project, I built a simple ETL (Extract, Transform, Load) pipeline using Python to collect real-time weather data from the OpenWeather API.

Project Overview

The pipeline performs the following steps:

Extract: Retrieved live weather data for multiple cities using the OpenWeather API.
Transform: Converted the JSON response into a clean, structured Pandas DataFrame by organizing the data, formatting columns, and preparing it for analysis.
Load: Saved the processed data into a CSV file for future analysis and reporting.
